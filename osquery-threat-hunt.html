Process & Execution Analysis

All Running Processes (Full Detail) BASELINE
Full snapshot of running processes — essential baseline for hunting anomalies.
SELECT pid, name, path, cmdline, uid, gid,
       start_time, parent, cwd()
FROM processes
ORDER BY start_time DESC;
Look for processes with no path, unusual cwd, or non-standard parent
Processes Spawned from Temp / Suspicious Dirs HIGH VALUE
Malware commonly executes from temp, AppData, or download directories.
SELECT pid, name, path, cmdline, parent
FROM processes
WHERE path LIKE '%\Temp\%'
   OR path LIKE '%\AppData\%'
   OR path LIKE '%\Downloads\%'
   OR path LIKE '%/tmp/%'
   OR path LIKE '%/var/tmp/%';
T1059 – Command & Scripting Interpreter | T1204 – User Execution
Unsigned / No-Path Processes (Windows) ANOMALY
Processes running with empty path may indicate process hollowing or injection.
SELECT p.pid, p.name, p.path, p.cmdline,
       p.parent, u.username
FROM processes p
JOIN users u ON p.uid = u.uid
WHERE p.path = ''
  AND p.name != '';
T1055 – Process Injection | Cross-ref with process memory maps
Suspicious Parent-Child Chains HIGH VALUE
Detects Office/browser spawning shells — classic phishing/macro delivery.
SELECT p.pid, p.name, p.path, p.cmdline,
       pp.name AS parent_name, pp.path AS parent_path
FROM processes p
JOIN processes pp ON p.parent = pp.pid
WHERE pp.name IN (
  'WINWORD.EXE', 'EXCEL.EXE', 'POWERPNT.EXE',
  'chrome.exe', 'firefox.exe', 'outlook.exe'
)
AND p.name IN (
  'cmd.exe', 'powershell.exe', 'wscript.exe',
  'cscript.exe', 'mshta.exe', 'bash.exe'
);
T1566.001 – Spearphishing | T1059.001 – PowerShell
Processes with Open File Handles FORENSICS
Correlate processes with open handles — useful for ransomware hunting.
SELECT p.pid, p.name, p.path,
       pof.path AS open_file
FROM processes p
JOIN process_open_files pof ON p.pid = pof.pid
WHERE pof.path LIKE '%.docx'
   OR pof.path LIKE '%.xlsx'
   OR pof.path LIKE '%.pdf'
ORDER BY p.name;
T1486 – Data Encrypted for Impact (Ransomware)
Masquerading Processes (Name Mismatch) MASQUERADE
Finds processes using system binary names running from non-system paths.
SELECT pid, name, path, cmdline
FROM processes
WHERE name IN (
  'svchost.exe', 'lsass.exe', 'csrss.exe',
  'explorer.exe', 'winlogon.exe'
)
AND path NOT LIKE '%\Windows\System32\%'
AND path NOT LIKE '%\Windows\SysWOW64\%'
AND path != '';
T1036 – Masquerading | Critical Windows process impersonation

Network Connections & Listeners

All Active Network Connections BASELINE
Full connection list with process attribution — start every hunt here.
SELECT p.pid, p.name, p.path,
       s.local_address, s.local_port,
       s.remote_address, s.remote_port,
       s.state, s.protocol
FROM process_open_sockets s
JOIN processes p ON s.pid = p.pid
WHERE s.state = 'ESTABLISHED'
ORDER BY s.remote_address;
Filter to ESTABLISHED for active sessions, LISTEN for backdoors
Listening on Unusual Ports BACKDOOR
Detects unexpected listeners that may indicate C2 implants or backdoors.
SELECT p.pid, p.name, p.path,
       s.local_address, s.local_port
FROM listening_ports s
JOIN processes p ON s.pid = p.pid
WHERE s.local_port NOT IN (
  80, 443, 22, 53, 3389,
  445, 135, 139, 5985
)
AND s.local_address = '0.0.0.0'
ORDER BY s.local_port;
T1049 – System Network Connections Discovery | T1571 – Non-Standard Port
DNS Cache Entries (Windows) DNS
Review recently resolved domains for C2 beaconing or DGA patterns.
SELECT name, type, flags,
       address, ttl
FROM dns_cache
WHERE name NOT LIKE '%.microsoft.com'
  AND name NOT LIKE '%.windows.com'
  AND name NOT LIKE '%.apple.com'
ORDER BY name;
T1568.002 – DGA | Look for high-entropy domain names
Outbound Connections to Rare External IPs C2 HUNT
Finds established connections to non-RFC1918 / non-loopback addresses.
SELECT p.name, s.remote_address,
       s.remote_port, s.local_port,
       p.cmdline, s.state
FROM process_open_sockets s
JOIN processes p ON s.pid = p.pid
WHERE s.remote_address NOT LIKE '10.%'
  AND s.remote_address NOT LIKE '192.168.%'
  AND s.remote_address NOT LIKE '172.16.%'
  AND s.remote_address != '127.0.0.1'
  AND s.state = 'ESTABLISHED'
  AND p.name NOT IN (
    'chrome.exe', 'firefox.exe', 'msedge.exe'
  );
T1071 – Application Layer Protocol | Pivot to threat intel feeds

Persistence Mechanisms

Startup Items & LaunchDaemons (macOS) PERSIST
Enumerates LaunchAgents/Daemons — primary macOS persistence locations.
SELECT name, path, program,
       program_arguments, run_at_load, disabled
FROM launchd
WHERE disabled = 0
  AND run_at_load = 1
ORDER BY path;
T1543.004 – Launch Daemon | Review ~/Library/LaunchAgents too
Windows Registry Run Keys PERSIST
Classic Windows persistence via HKCU/HKLM Run registry keys.
SELECT name, path, data, type
FROM registry
WHERE path LIKE (
  'HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\'
  'Windows\CurrentVersion\Run%'
)
OR path LIKE (
  'HKEY_CURRENT_USER\SOFTWARE\Microsoft\'
  'Windows\CurrentVersion\Run%'
);
T1547.001 – Registry Run Keys | Check data field for suspicious paths
Cron Jobs & Scheduled Tasks (Linux) SCHEDULED
Lists all cron jobs — common Linux/macOS persistence mechanism.
SELECT event, minute, hour,
       day_of_month, month, day_of_week,
       command, path
FROM crontab
ORDER BY path;
T1053.003 – Cron | Watch for base64-encoded commands in cmdline
Windows Scheduled Tasks SCHEDULED
Full list of scheduled tasks including action commands for triage.
SELECT name, action, path,
       enabled, last_run_time,
       next_run_time, status
FROM scheduled_tasks
WHERE enabled = 1
  AND action LIKE '%powershell%'
   OR action LIKE '%cmd%'
   OR action LIKE '%wscript%'
   OR action LIKE '%mshta%';
T1053.005 – Scheduled Task | Flag tasks running interpreter binaries
Startup Programs (Windows) PERSIST
Startup folder entries and WMI subscriptions for persistence.
SELECT name, path, args, status, source
FROM startup_items
ORDER BY name;
T1547 – Boot/Logon Autostart | Cross with file hash lookup
Browser Extensions (Chrome) BROWSER
Extensions can be used for data exfiltration or persistent access.
SELECT uid, name, identifier,
       version, description, permissions,
       path
FROM chrome_extensions
WHERE permissions LIKE '%nativeMessaging%'
   OR permissions LIKE '%debugger%'
   OR permissions LIKE '%webRequest%';
T1176 – Browser Extensions | High-risk permissions indicate risk

Users, Auth & Privilege Escalation

Recently Created User Accounts PRIVESC
New accounts created by attackers for persistence or lateral movement.
SELECT uid, gid, username,
       description, directory, shell,
       uuid
FROM users
WHERE uid >= 1000
  AND shell != '/usr/sbin/nologin'
  AND shell != '/bin/false'
ORDER BY uid DESC;
T1136 – Create Account | Shell != nologin means interactive login
Sudoers & Privileged Groups PRIVESC
Audit who has sudo/wheel access — common escalation target.
SELECT g.groupname, u.username,
       u.uid, u.shell
FROM groups g
JOIN user_groups ug ON g.gid = ug.gid
JOIN users u ON ug.uid = u.uid
WHERE g.groupname IN (
  'sudo', 'wheel', 'admin',
  'root', 'Administrators'
);
T1078 – Valid Accounts | T1548 – Abuse Elevation Control Mechanism
SSH Authorized Keys BACKDOOR
Unauthorized public keys in authorized_keys grant persistent SSH access.
SELECT u.username, u.uid,
       ak.key_file, ak.algorithm,
       ak.comment, ak.key
FROM users u
JOIN authorized_keys ak
  ON ak.uid = u.uid
ORDER BY u.username;
T1098.004 – SSH Authorized Keys | Any unrecognized key is suspect
SUID / SGID Binaries PRIVESC
SUID binaries run as owner — attackers plant them for privilege escalation.
SELECT path, filename, size,
       permissions, uid, gid,
       sha256
FROM file
WHERE (permissions LIKE '%s%'
  OR permissions LIKE '%S%')
  AND path NOT LIKE '/usr/bin/%'
  AND path NOT LIKE '/bin/%'
ORDER BY path;
T1548.001 – Setuid/Setgid | Unexpected SUID outside /usr is critical

File System & Artifact Hunting

Recently Modified System Files TAMPER
Detects recent modifications to sensitive system directories.
SELECT path, filename, size,
       mtime, atime, ctime,
       permissions, uid
FROM file
WHERE (path LIKE '/etc/%'
   OR path LIKE '/bin/%'
   OR path LIKE '/usr/bin/%')
  AND mtime > (strftime('%s', 'now') - 86400)
ORDER BY mtime DESC;
T1565 – Data Manipulation | mtime-86400 = last 24hrs
Hash Lookup — Known-Bad IOCs IOC
Match process binary hashes against a known-bad IOC list.
SELECT p.pid, p.name, p.path,
       h.sha256, h.md5
FROM processes p
JOIN hash h ON p.path = h.path
WHERE h.sha256 IN (
  -- Replace with your IOC hashes
  'abc123deadbeef...',
  'def456cafebabe...'
);
Integrate with VirusTotal or MISP feeds for automation
Executable Files in Home Dirs ANOMALY
Executables in /home or /Users are unusual and warrant investigation.
SELECT path, filename, size,
       mtime, permissions, uid
FROM file
WHERE (path LIKE '/home/%'
   OR path LIKE '/Users/%')
  AND (permissions LIKE '%x%')
  AND filename NOT LIKE '%.sh'
ORDER BY mtime DESC;
T1204 – User Execution | ELF/PE binaries in home dirs are high-risk
Installed Software Inventory INVENTORY
Full software list for baselining and detecting unapproved tools.
SELECT name, version, source,
       install_location, install_date
FROM programs            -- Windows
UNION ALL
SELECT name, version, source,
       '', install_time
FROM deb_packages        -- Debian/Ubuntu
ORDER BY name;
Flag hacking tools: nmap, netcat, mimikatz, etc.

Quick Reference — Tables & Tips

Essential OSquery Tables

  • processes — running process details
  • process_open_sockets — network connections per PID
  • listening_ports — active listeners
  • users / groups — local accounts
  • file / hash — filesystem + crypto hashes
  • registry — Windows registry
  • startup_items / launchd — autorun entries
  • scheduled_tasks / crontab — task scheduling
  • dns_cache — resolved hostnames
  • authorized_keys — SSH keys
  • chrome_extensions — browser addons
  • syslog / system_info — logs & host metadata

OSquery Key Concepts

  • osqueryi — interactive SQL shell for ad-hoc queries
  • osqueryd — daemon mode with scheduled packs
  • Packs — JSON query bundles (e.g. incident-response pack)
  • FIM — File Integrity Monitoring via file_events
  • ATCs — Auto Table Construction for custom data
  • Extensions — Go/Python plugins to add tables
  • --json flag — output as JSON for SIEM ingestion
  • JOIN — correlate tables, e.g. processes + sockets
  • WHERE epoch — use strftime('%s','now') for time filters
  • Differential mode — detect changes between snapshots
